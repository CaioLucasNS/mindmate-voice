## ‚öõÔ∏è React & React Native

### Components

#### ‚úÖ **Correct**

```typescript
// Arrow function for components
const HomeScreen = () => {
  const { theme } = useThemeApp();

  return (
    <View style={styles.container}>
      <Text>Hello World</Text>
    </View>
  );
};

```

#### ‚ùå **Incorrect**

```typescript
// Function declaration (not allowed)

  return <View><Text>Hello World</Text></View>;
}
```

### Hooks

#### ‚úÖ **Correct**

```typescript
// useCallback for functions passed as props
const handlePress = useCallback(() => {
  onPress();
}, [onPress]);

// useMemo for expensive calculations
const expensiveValue = useMemo(() => {
  return heavyCalculation(data);
}, [data]);

// useEffect with correct dependencies
useEffect(() => {
  fetchData();
}, [fetchData]);
```

#### ‚ùå **Incorrect**

```typescript
// Function recreated on every render
const handlePress = () => {
  onPress();
};

// Calculation executed on every render
const expensiveValue = heavyCalculation(data);

// useEffect without dependencies
useEffect(() => {
  fetchData();
}, []);
```

### Context Providers

#### ‚úÖ **Correct**

```typescript
const ThemeProvider = ({ children }: { children: React.ReactNode }) => {
  const [theme, setTheme] = useState('light');

  const toggleTheme = useCallback(() => {
    setTheme(prev => prev === 'light' ? 'dark' : 'light');
  }, []);

  const contextValue = useMemo(() => ({
    theme,
    toggleTheme,
  }), [theme, toggleTheme]);

  return (
    <ThemeContext.Provider value={contextValue}>
      {children}
    </ThemeContext.Provider>
  );
};
```

#### ‚ùå **Incorrect**

```typescript
// Object recreated on every render
<ThemeContext.Provider value={{ theme, toggleTheme }}>
  {children}
</ThemeContext.Provider>
```

## üé® Styles

### StyleSheet vs Inline Styles

#### ‚úÖ **Correct**

```typescript
// File: styles.ts

  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  text: {
    fontSize: 16,
    color: colors.textPrimary,
  },
});

const Component = () => (
  <View style={styles.container}>
    <Text style={styles.text}>Hello</Text>
  </View>

```

#### ‚ùå **Incorrect**

```typescript
// Inline styles (not allowed)
const Component = () => (
  <View style={{ flex: 1, justifyContent: 'center' }}>
    <Text style={{ fontSize: 16, color: '#333' }}>Hello</Text>
  </View>
);
```

## üîß TypeScript

### Interfaces and Types

#### ‚úÖ **Correct**

```typescript

  title: string;
  onPress: () => void;
  variant?: 'primary' | 'secondary';
  disabled?: boolean;
}
```

```typescript

const data: any = response.data;

const Component = ({ title, onPress }: { title: string; onPress: () => void }) => {
  // ...
```

## üîí Security

### Input Validation

```typescript
// Validate all inputs
const validateEmail = (email: string): boolean => {
  const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return emailRegex.test(email);
};

// Sanitize data
const sanitizeInput = (input: string): string => {
  return input.replace(/[<>]/g, '');
};
```

### Secure Storage

```typescript
// Use secure storage for sensitive data
import * as SecureStore from 'expo-secure-store';

const storeToken = async (token: string) => {
  await SecureStore.setItemAsync('auth_token', token);

const getToken = async (): Promise<string | null> => {
  return await SecureStore.getItemAsync('auth_token');
```

## üéØ Best Practices Summary

### Do's

- ‚úÖ Use arrow functions for components
- ‚úÖ Implement useCallback and useMemo appropriately
- ‚úÖ Use StyleSheet for styles
- ‚úÖ Follow TypeScript strict mode
- ‚úÖ Write comprehensive tests
- ‚úÖ Add accessibility props
- ‚úÖ Validate all inputs
- ‚úÖ Use semantic naming
- ‚úÖ Document complex logic
- ‚úÖ Follow Clean Architecture principles

### Don'ts

- ‚ùå Use function declarations for components
- ‚ùå Create functions on every render
- ‚ùå Use inline styles
- ‚ùå Use 'any' type
- ‚ùå Skip writing tests
- ‚ùå Ignore accessibility
- ‚ùå Trust user input
- ‚ùå Use unclear variable names
- ‚ùå Leave code undocumented
- ‚ùå Mix concerns across layers
